## Local development

### Run the app

Make sure you have the current vesion of [Node](https://nodejs.org/en/) and [Yarn](https://classic.yarnpkg.com/en/)

Install all dependencies

```bash
$ yarn
```

Run the app

```bash
yarn dev
```

### Access the app

The React web app is served at http://localhost:3000
